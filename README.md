# KeenKeeper

KeenKeeper is a simple web application for managing friendships and keeping track of interactions it helps users remember when they last contacted someone and stay connected more consistently.

## Description
This project allows users to view a list of friends, track the last time they interacted, and log new interactions such as calls texts or video chats it also includes a timeline to visualize communication history.
## Technologies Used

* Next.js
* React
* Tailwind CSS
* Recharts

## Key Features

1. Track days since last contact with each friend
2. Log interactions like call, text, or video
3. View interaction history in a timeline
